# 布局系统

布局系统位于 `core/layout.h`，当前是 header-only 的纯计算模块，不依赖 OpenGL。

## 1. 容器类型

```cpp
enum class LayoutType {
    Row,
    Column,
    Stack
};
```

- Row：子元素横向排列
- Column：子元素纵向排列
- Stack：子元素叠加

## 2. 对齐枚举

```cpp
enum class Align {
    START,
    CENTER,
    END
};
```

每个容器有：

- `mainAlign`：主轴对齐
- `crossAlign`：交叉轴对齐

规则：

- Row：主轴是 x，crossAlign 控制垂直方向
- Column：主轴是 y，crossAlign 控制水平方向
- Stack：当前用 mainAlign 控制 y，用 crossAlign 控制 x

## 3. 尺寸模式

```cpp
enum class SizeMode {
    Fixed,
    WrapContent,
    Fill
};
```

使用 `SizeValue`：

```cpp
node.setWidth(core::SizeValue::fixed(240.0f));
node.setHeight(core::SizeValue::wrapContent());
node.setWidth(core::SizeValue::fill());
```

便捷写法：

```cpp
node.setFixedSize(240.0f, 70.0f);
```

## 4. Margin

```cpp
core::EdgeInsets margin;
margin.left = 8.0f;
margin.top = 4.0f;
margin.right = 8.0f;
margin.bottom = 4.0f;

node.setMargin(margin);
```

统一 margin：

```cpp
node.setMargin(core::EdgeInsets::all(8.0f));
```

## 5. Spacing

`spacing` 只对 Row / Column 有意义：

```cpp
container.setSpacing(8.0f);
```

- Row：子元素之间的水平间距
- Column：子元素之间的垂直间距

## 6. 核心函数

```cpp
root.measure(availableWidth, availableHeight);
root.layout(x, y);
```

`measure()` 计算每个节点的最终尺寸。  
`layout(x, y)` 计算每个节点的最终位置。

结果通过：

```cpp
node.frame()
```

读取：

```cpp
const core::LayoutRect& frame = node.frame();
```

## 7. Row 示例

```cpp
core::Node row(core::LayoutType::Row);
row.setFixedSize(400.0f, 100.0f);
row.setMainAlign(core::Align::CENTER);
row.setCrossAlign(core::Align::CENTER);
row.setSpacing(12.0f);

auto* a = row.addChild();
a->setFixedSize(80.0f, 40.0f);

auto* b = row.addChild();
b->setFixedSize(120.0f, 60.0f);

row.measure(400.0f, 100.0f);
row.layout(0.0f, 0.0f);
```

Row 会：

1. 计算所有子节点 outer width 总和
2. 加上 spacing
3. 根据 mainAlign 决定 x 起点
4. 根据 crossAlign 决定每个子节点 y

## 8. Column 示例

```cpp
core::Node column(core::LayoutType::Column);
column.setFixedSize(360.0f, 260.0f);
column.setMainAlign(core::Align::CENTER);
column.setCrossAlign(core::Align::CENTER);
column.setSpacing(8.0f);

auto* title = column.addChild();
title->setFixedSize(300.0f, 38.0f);

auto* button = column.addChild();
button->setFixedSize(240.0f, 70.0f);

column.measure(360.0f, 260.0f);
column.layout(100.0f, 100.0f);
```

Column 会：

1. 计算 totalHeight
2. 根据 mainAlign 决定 y 起点
3. 根据 crossAlign 决定每个子节点 x

## 9. Stack 示例

```cpp
core::Node root(core::LayoutType::Stack);
root.setFixedSize(width, height);
root.setMainAlign(core::Align::CENTER);
root.setCrossAlign(core::Align::CENTER);

auto* card = root.addChild(core::LayoutType::Column);
card->setFixedSize(360.0f, 260.0f);

root.measure(width, height);
root.layout(0.0f, 0.0f);
```

Stack 当前会把所有子元素叠放在容器内，并按对齐规则计算位置。

## 10. Demo 中的使用方式

`app/demo.cpp` 中：

- 用根 `Stack` 居中卡片
- 卡片内部用 `Column`
- title / subtitle / button 都作为子节点
- layout 得到逻辑坐标
- 再乘 `dpiScale` 转为 framebuffer 像素坐标

核心流程：

```cpp
const DemoFrames logical = measureDemo(logicalWidth, logicalHeight);
const core::LayoutRect panelFrame = toPixels(logical.panel, dpiScale);
panel.setFrame(panelFrame.x, panelFrame.y, panelFrame.width, panelFrame.height);
```

## 11. 当前限制

- 没有 flex-grow / flex-shrink
- Fill 对子节点分配还比较基础
- 没有 padding
- 没有 min/max width/height
- 没有布局脏标记缓存
- 组件还没有直接继承 Node

后续可以把布局和组件绑定成真正的 Widget Tree。
