# 基础图元与文本图元

本文档说明 `core/primitive.h` 和 `core/text.h/.cpp` 中的底层绘制能力。

## 1. 基础类型

`core::Vec2`

```cpp
struct Vec2 {
    float x;
    float y;
};
```

`core::Color`

```cpp
struct Color {
    float r;
    float g;
    float b;
    float a;
};
```

`core::Rect`

```cpp
struct Rect {
    float x;
    float y;
    float width;
    float height;

    bool contains(double pointX, double pointY) const;
};
```

当前坐标体系使用 framebuffer 像素坐标，左上角为原点，y 向下。

## 2. RoundedRectPrimitive

基础圆角矩形图元：

```cpp
core::RoundedRectPrimitive rect(x, y, width, height);
rect.initialize();
rect.render(windowWidth, windowHeight);
rect.destroy();
```

支持属性：

- bounds：`setBounds(x, y, width, height)`
- color：`setColor(...)`
- opacity：`setOpacity(...)`
- corner radius：`setCornerRadius(...)`
- gradient：`setGradient(...)`
- border：`setBorder(...)`
- shadow：`setShadow(...)`
- translate：`setTranslate(...)`
- scale：`setScale(...)`
- rotate：`setRotate(...)`
- transform origin：`setTransformOrigin(...)`

## 3. Gradient

```cpp
core::Gradient gradient;
gradient.enabled = true;
gradient.start = {0.10f, 0.12f, 0.16f, 1.0f};
gradient.end = {0.05f, 0.07f, 0.10f, 1.0f};
gradient.direction = core::GradientDirection::Vertical;
```

当前支持：

- Horizontal
- Vertical

## 4. Border

```cpp
core::Border border;
border.width = 1.0f;
border.color = {0.23f, 0.29f, 0.38f, 1.0f};
```

边框在 fragment shader 中根据圆角矩形 SDF 距离计算。

## 5. Shadow

```cpp
core::Shadow shadow;
shadow.enabled = true;
shadow.offset = {0.0f, 8.0f};
shadow.blur = 26.0f;
shadow.spread = 0.0f;
shadow.color = {0.0f, 0.0f, 0.0f, 0.26f};
```

阴影实现是分层近似：

- ambient 大软影
- mid 中层软影
- contact 近距离接触影

它不是完整 CSS blur，但比单层黑边更接近 EUI 风格的盒阴影。

## 6. Transform

```cpp
rect.setTranslate(10.0f, 0.0f);
rect.setScale(1.0f, 1.0f);
rect.setRotate(0.1f);
rect.setTransformOrigin(0.5f, 0.5f);
```

`transform-origin` 使用 0 到 1 的相对值：

- `{0, 0}` 左上
- `{0.5, 0.5}` 中心
- `{1, 1}` 右下

注意：当前事件 hit-test 仍基于未旋转矩形 bounds。后续如果要完整支持旋转命中测试，需要把 pointer 反变换到局部坐标。

## 7. TextPrimitive

文本图元在：

- `core/text.h`
- `core/text.cpp`

使用 `3rd/stb_truetype.h` 的 SDF 生成能力。

基础用法：

```cpp
core::TextPrimitive text(40.0f, 40.0f);

core::TextStyle style;
style.text = "Hello GUI";
style.fontSize = 28.0f;
style.color = {1, 1, 1, 1};
style.maxWidth = 300.0f;
style.wrap = true;

text.setStyle(style);
text.initialize();
text.render(windowWidth, windowHeight);
text.destroy();
```

## 8. TextStyle 属性

```cpp
struct TextStyle {
    std::string text;
    std::string fontFamily;
    float fontSize;
    int fontWeight;
    Color color;
    float maxWidth;
    bool wrap;
    HorizontalAlign horizontalAlign;
    VerticalAlign verticalAlign;
    float lineHeight;
};
```

支持：

- text：文本内容
- fontFamily：字体路径或常见字体名
- fontSize：字号
- fontWeight：粗细
- color：颜色
- maxWidth：最大宽度
- wrap：是否换行
- horizontalAlign：Left / Center / Right
- verticalAlign：Top / Center / Bottom
- lineHeight：行高，默认可用 `fontSize * 1.2`

## 9. 字体内存策略

当前策略：

- 字体文件数据按路径共享缓存
- 每个 `TextPrimitive` 仍持有自己的 glyph atlas 纹理
- atlas 默认 `512x512`
- 不再保留 CPU atlas 副本

因此多个文本使用同一字体时，不会重复加载字体文件数据；但 atlas 目前还没有全局共享。后续优化方向是 `FontAtlas` / `TextRenderer` 统一管理。

## 10. 当前限制

- 文本 atlas 不会自动扩容
- 暂未做 fallback font
- 暂未做 kerning
- 暂未做复杂文字排版
- 每个 TextPrimitive 一个 OpenGL shader/VAO/VBO/texture

这版足够做轻量 GUI 的基础文本，但还不是最终高性能文本系统。
